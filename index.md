# なぜこのブログはこんなに変なんですか？

このデザインはあえて違いを出しています。というのも、視覚的な個性は強く心に残る印象を生むと信じているからです。とはいえ、型破りな見た目の裏側でも、主眼はあくまで中身――すなわち、まっすぐな思いと、読者の心に触れ、インスピレーションを与えられる伝え方にあります (˶ᵔ ᵕ ᵔ˶)


# あなたの投稿をすべて見るにはどこを見ればいいですか？

このブログのすべてのコンテンツを探索したい方のために、各ページのすべての記事をまとめたアーカイブページを用意しました。新しい投稿は公開されるたびに自動的に表示されます。スムーズに操作できるよう、右下の矢印ボタンをご利用ください。直感的にナビゲートできます。


# ブログコンポーネントについて

このブログは、デザインと機能の要件に合わせてカスタマイズした Cayman Custom テンプレートを使用しています。ビジュアル要素については、背景画像の大半を Pixiv から使用していますが、ほかの出所のものも一部含まれます。すべてのクレジットと出典情報は、メインの GitHub リポジトリで完全に公開しています。
