# сЃесЃЏсЃ╝!!!

```
РађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРБђРБђРБђРађРађРађРађРађРађРађРађРађРађ
РађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРБђРађРађРађРађРађРађРађРађРађРађРбђРА┤РаіРаЅРађРаѕРаЎРбдРађРађРађРађРађРађРађРађ
РађРађРађРађРађРађРађРађРађРађРађРађРађРађРБ┤РБ┐РБђРађРађРађРађРађРађРађРађРА░РаІРАаРаќРаІРаЅРаЏРбиРБХРбиРАђРађРађРађРађРађРађ
РађРађРађРађРађРађРађРађРађРађРађРађРБаРађРаўРб┐РБЕРБДРађРађРађРађРађРађРА╝РБАРајРађРађРађРађРађРађРа╣РБЇРБДРађРађРађРађРађРађ
РађРађРађРађРађРађРађРађРађРађРађРб┤РБ┐РАХРађРађРаЅРаІРађРађРађРађРађРБИРА┐РаЂРађРађРађРађРађРађРађРађРаИРБ┐РађРађРађРађРађРађ
РађРађРађРађРађРађРађРађРађРађРађРађРаЎРаЃРађРађРбђРБаРа┤РањРбЏРБ╗РБЏРБ┐РБДРБцРБђРАђРађРађРађРађРађРађРбђРБ┐РаЃРађРађРађРађРађ
РађРађРађРађРађРађРађРађРађРађРађРађРбђРБаРБ┤РаъРаЅРађРаљРа╗РаЪРаЏРбЏРБиРађРађРађРаЅРаЊРбдРАђРађРађРађРаѕРаІРађРађРађРађРађРађ
РађРађРађРађРађРађРађРаўРа╗РбХРБъРБІРБАРаъРаЂРБёРБђРБђРБђРБаРАХРа┐РаЏРбІРађРађРађРађРбцРАђРаѕРббРАђРађРађРађРбђРБЙРБДРађРађРађ
РађРађРађРађРађРађРађРађРбђРА┤РбџРА┐РаЂРађРБаРБЙРБйРаиРбдРБцРацРбђРБ┤РАЪРађРАёРБ┐РађРаѕРа╗РАёРАёРа╣РАёРађРађРБЙРБЙРаІРађРбђРађ
РађРађРађРађРађРађРБцРаъРбЏРБйРАЪРаЊРбХРБџРБІРАЎРа▓РбЙРБГРБ┤РбХРБ┐РАЪРађРБёРБ┐Ра╗РБДРађРађРбиРаИРАєРа╣РАёРађРађРаЂРађРб┤РБ┐РБи
РађРађРађРађРађРађРађРбђРАЙРБ╗РаЂРбаРбђРБ┐РБйРа┐РаќРаІРаЪРађРБ║РБЙРБЂРБХРа┐РаЅРађРаИРАєРађРБ┐РађРБиРађРб╗РађРађРађРађРађРаЏРаЂ
РађРБаРБХРАёРађРађРбђРБЙРБ┐РАЄРађРБ╝РбИРБДРБ┐РађРа▓РБёРађРањРаЏРаІРаЂРађРађРађРбђРБаРБДРбаРБ┐РађРБ┐РАёРбИРАєРађРађРађРађРађРађ
РађРаЅРа┐РаЂРађРбђРА╝РБ»РБ╝РАќРбаРАЄРБИРаЂРаѕРбаРБцРБїРаЎРађРађРађРађРађРаЏРБЏРБЅРбђРБДРаъРБ┐РађРб╣РБ┐РБеРБДРађРађРађРађРађРађ
РбђРађРбђРБаРБ┤РБФРБ╝РБ┐РБйРбБРБ╝РБиРБЄРБХРБ┐РБ┐РБ┐РБ┐РаЄРађРађРађРађРађРБХРБ┐РБ┐РБЙРБДРБИРАЄРађРБ┐РБ┐РбИРБиРБиРБцРБђРАђРађРађ
РаѕРаЅРаЅРаЅРбЂРБ╝РБ┐РБ┐РБ┐РБЙРБ┐РБ┐РА┐РБ┐РБ┐РБ┐РБ┐РБЈРађРађРађРађРађРађРбІРБйРБ┐РБ┐РБ┐РБ┐РАЄРађРбИРБиРб╝РБЪРБєРаЅРаЅРаЂРађРађ
РађРађРађРБ┤РБ┐РАЙРбњРБ┐РаЏРаЃРБ┐РБЈРбБРАйРБ┐РБ╗РА┐РаЃРађРађРађРађРађРађРаўРБ┐РА┐РБ┐РА┐Рб╗РаЄРађРаИРБ┐РАєРб┐РБјРбдРађРађРађРађ
РађРађРБ╝РА┐РБ┐РБЄРБЙРАЈРађРбИРБ┐РБ┐РАІРАаРБбРАєРађРађРађРађРађРађРађРађРБХРБЪРБ┐РБ┐РађРбИРБдРађРа░РБ┐РБДРаѕРБ┐РБ┐РБЄРађРађРађ
РађРађРб╗РађРБ┐РБ┐РБ┐РАЄРађРаИРБ┐РБ┐РБ╗РаќРаІРађРађРађРааРаќРањРа▓РБёРађРаЂРаўРАЪРаЈРаџРбИРБ┐РАђРађРб╗РБ┐РађРБ┐РБ┐РаЪРађРађРађ
```

сЂЊсѓїсЂ»сђЂжБЙсѓЅсЂфсЂёсѓбсЃЌсЃГсЃ╝сЃЂсЂеУфат«ЪсЂфУфъсѓітЈБсЂДсЂіт▒ісЂЉсЂЎсѓІсђЂсЂћсЂЈТЎ«жђџсЂ«сЃќсЃГсѓ░сЂДсЂЎсђѓ


## сЂфсЂюсЂЊсЂ«сЃќсЃГсѓ░сЂ»сЂЊсѓЊсЂфсЂФтцЅсЂфсѓЊсЂДсЂЎсЂІ№╝Ъ

сЂЊсЂ«сЃЄсѓХсѓцсЃ│сЂ»сЂѓсЂѕсЂджЂЋсЂёсѓњтЄ║сЂЌсЂдсЂёсЂЙсЂЎсђѓсЂесЂёсЂєсЂ«сѓѓсђЂУдќУдџуџёсЂфтђІТђДсЂ»т╝исЂЈт┐ЃсЂФТ«ІсѓІтЇ░У▒АсѓњућЪсѓђсЂеС┐АсЂўсЂдсЂёсѓІсЂІсѓЅсЂДсЂЎсђѓсЂесЂ»сЂёсЂѕсђЂтъІуа┤сѓісЂфУдІсЂЪуЏ«сЂ«УБЈтЂ┤сЂДсѓѓсђЂСИ╗ую╝сЂ»сЂѓсЂЈсЂЙсЂДСИГУ║ФРђЋРђЋсЂЎсЂфсѓЈсЂАсђЂсЂЙсЂБсЂЎсЂљсЂфТђЮсЂёсЂесђЂУфГУђЁсЂ«т┐ЃсЂФУДдсѓїсђЂсѓцсЃ│сѓ╣сЃћсЃгсЃ╝сѓисЃДсЃ│сѓњСИјсЂѕсѓЅсѓїсѓІС╝ЮсЂѕТќ╣сЂФсЂѓсѓісЂЙсЂЎ (╦Хрхћ рхЋ рхћ╦Х)


## сЂѓсЂфсЂЪсЂ«ТіЋуе┐сѓњсЂЎсЂ╣сЂдУдІсѓІсЂФсЂ»сЂЕсЂЊсѓњУдІсѓїсЂ░сЂёсЂёсЂДсЂЎсЂІ№╝Ъ

сЂЊсЂ«сЃќсЃГсѓ░сЂ«сЂЎсЂ╣сЂдсЂ«сѓ│сЃ│сЃєсЃ│сЃёсѓњТјбу┤бсЂЌсЂЪсЂёТќ╣сЂ«сЂЪсѓЂсЂФсђЂтљёсЃџсЃ╝сѓИсЂ«сЂЎсЂ╣сЂдсЂ«УеўС║ІсѓњсЂЙсЂесѓЂсЂЪсѓбсЃ╝сѓФсѓцсЃќсЃџсЃ╝сѓИсѓњућеТёЈсЂЌсЂЙсЂЌсЂЪсђѓТќ░сЂЌсЂёТіЋуе┐сЂ»тЁгжќІсЂЋсѓїсѓІсЂЪсЂ│сЂФУЄфтІЋуџёсЂФУАеуц║сЂЋсѓїсЂЙсЂЎсђѓсѓ╣сЃасЃ╝сѓ║сЂФТЊЇСйюсЂДсЂЇсѓІсѓѕсЂєсђЂтЈ│СИІсЂ«уЪбтЇ░сЃюсѓ┐сЃ│сѓњсЂћтѕЕућесЂЈсЂасЂЋсЂёсђѓуЏ┤ТёЪуџёсЂФсЃісЃЊсѓ▓сЃ╝сЃѕсЂДсЂЇсЂЙсЂЎсђѓ


## сЃќсЃГсѓ░сѓ│сЃ│сЃЮсЃ╝сЃЇсЃ│сЃѕсЂФсЂцсЂёсЂд

сЂЊсЂ«сЃќсЃГсѓ░сЂ»сђЂсЃЄсѓХсѓцсЃ│сЂеТЕЪУЃйсЂ«УдЂС╗ХсЂФтљѕсѓЈсЂЏсЂдсѓФсѓ╣сѓ┐сЃъсѓцсѓ║сЂЌсЂЪ Cayman Custom сЃєсЃ│сЃЌсЃгсЃ╝сЃѕсѓњСй┐ућесЂЌсЂдсЂёсЂЙсЂЎсђѓсЃЊсѓИсЃЦсѓбсЃФУдЂу┤асЂФсЂцсЂёсЂдсЂ»сђЂУЃїТЎ»ућ╗тЃЈсЂ«тцДтЇісѓњ Pixiv сЂІсѓЅСй┐ућесЂЌсЂдсЂёсЂЙсЂЎсЂїсђЂсЂ╗сЂІсЂ«тЄ║ТЅђсЂ«сѓѓсЂ«сѓѓСИђжЃетљФсЂЙсѓїсЂЙсЂЎсђѓсЂЎсЂ╣сЂдсЂ«сѓ»сЃгсѓИсЃЃсЃѕсЂетЄ║тЁИТЃЁта▒сЂ»сђЂсЃАсѓцсЃ│сЂ« GitHub сЃфсЃЮсѓИсЃѕсЃфсЂДт«їтЁесЂФтЁгжќІсЂЌсЂдсЂёсЂЙсЂЎсђѓ


## т░єТЮЦсЂФтљЉсЂЉсЂЪуЏ«ТеЎ

сЂєсЃ╝сѓЊРђдуЏ«уџёсЂФсЂцсЂёсЂдсЂ»сЂЙсЂаТђЮсЂёсЂцсЂёсЂдсЂёсЂЙсЂЏсѓЊсЂїсђЂС╗ісЂ«сЂесЂЊсѓЇсЂ»сѓхсѓцсЃѕсЂ«сЃЄсѓХсѓцсЃ│сѓњтцЅТЏ┤сЂЎсѓІсЂЊсЂесЂЌсЂІсЂДсЂЇсЂџсђЂсЂЊсЂАсѓЅсЂДУЄфтѕєсЂ«сЃфсЃЮсѓИсЃѕсЃфсѓѓтбЌсѓёсЂЌсЂдсЂёсЂЈсЂцсѓѓсѓісЂДсЂЎсђѓ
