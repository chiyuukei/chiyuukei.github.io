# сЃесЃЏсЃ╝!!!
```
РађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРбђРБђРБђРБаРацРАцРБёРБђРБђРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађ
РађРађРађРађРађРађРађРађРађРађРађРБђРБцРБ┤РБХРБХРаъРаІРбАРађРађРађРбаРАђРађРаЅРаЎРа▓РбцРБђРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађ
РађРађРађРађРађРађРађРађРбђРБ┤РбъРБЅРА┤РбЂРАЙРаЂРађРађРбИРАєРађРађРађРаЅРа│РБдРАђРађРаѕРаІРа│РБцРАђРађРађРађРађРађРађРађРађРађРађРађРађРађ
РађРађРађРађРађРађРбђРА┤РбЪРА┐РаІРаѕРађРБ╝РаЂРађРађРађРаѕРб│РАђРађРађРађРађРађРаЎРбХРБёРађРађРађРаЎРБдРАђРађРађРађРађРађРађРађРађРађРађРађ
РађРађРађРађРађРБаРаЪРбёРАъРађРађРађРбеРАЈРађРађРађРађРађРаѕРбиРађРађРађРађРађРађРађРаЎРбдРАђРађРаѓРаѕРб│РАёРађРађРађРађРађРађРађРађРађРађ
РађРађРађРађРБаРа┐РбђРАъРађРађРађРађРб╗РАЄРађРађРађРБђРБђРбђРАѕРБЄРађРађРађРађРађРађРађРађРа│РАёРађРађРађРб╗РАђРађРађРађРађРађРађРађРађРађ
РађРађРађРбаРаЪРађРБЙРађРађРађРаљРа▓РБ╝РАЄРбђРађРБЙРАЈРб╣РБиРАђРб╣РАёРађРађРађРађРађРађРађРађРа╣РАєРађРађРађРбиРАђРађРађРађРађРађРађРађРађ
РађРађРађРБЙРађРбИРаЄРађРбђРБђРБђРБаРа┐Ра┐РаЙРаиРбЙРАЄРбИРАЪРаІРаЏРаІРаЅРаЅРаЎРаЊРањРа▓РаХРадРБ╝РбйРАёРађРађРаўРБЄРађРађРађРађРађРађРађРађ
РађРађРБ╝РаІРађРБ┐РаЏРаІРаЂРађРађРађРађРађРађРађРађРаЏРаЏРаЃРађРађРађРађРађРађРађРађРађРађРађРб┐РааРб┐РАђРађРађРб╗РАђРађРађРађРађРађРађРађ
РађРбђРБ┐РађРбаРБ┐РађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРбИРађРаѕРАиРбёРађРбИРБЄРађРађРађРађРађРађРађ
РађРбИРаЂРаИРб╣Рб╣РађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРбИРБХРађРб╣РАђРађРаѕРБ┐РађРађРађРађРађРађРађ
РађРБ╝РађРађРБ┐РБйРАєРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРаеРБ┐РбдРАюРбиРаёРађРБ┐РАёРађРађРађРађРађРађ
РбђРАЈРађРађРБ┐Рб║РАЄРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРаѕРА┐РађРађРађРађРађРб╗РАЄРађРађРађРађРађРађ
РбИРАЄРађРађРБ┐Рб╝РАЄРађРађРађРађРађРбђРађРађРађРађРађРађРађРађРађРађРађРАђРађРађРађРађРађРађРађРБЄРађРађРађРађРађРбИРАЄРађРађРађРађРађРађ
РБЙРБ┐РађРАђРб╣РБ╝РАЄРађРађРађРађРађРБ┐РБ┐РаєРађРађРаљРањРањРањРађРађРб╝РБ┐РАиРађРађРађРађРађРађРАЄРађРађРађРађРађРбИРАЄРађРађРађРађРађРађ
РБ┐РБ┐РБёРбБРаИРБ┐РАЄРађРађРађРађРађРаѕРаЂРађРађРађРађРађРађРађРађРађРађРаЅРаЂРађРађРађРађРађРађРАЄРађРаѕРб│РађРађРбИРАЄРађРађРађРађРађРађ
РБ┐РАЄРБЈРАёРБДРб║РБЄРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРАЄРађРађРбИРађРађРБИРАЄРађРађРађРађРађРађ
РБ┐РБ┐РБ┐РАЃРаѕРаѕРБ┐РађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРАЄРб░РађРбИРађРађРБЙРАЄРађРађРађРађРађРађ
Рб╣РБДРб╗РАєРађРађРб╗РађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРАЄРбИРађРбИРађРађРбИРАЄРађРађРађРађРађРађ
РбИРБ┐РбИРАЄРбаРБђРа╗РАЄРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРАЄРА┐РађРбИРађРађРб║РБиРађРађРађРађРађРађ
РаѕРБ┐РБєРАЄРађРб╣РађРАЄРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРађРБДРАЄРађРбИРађРађРбИРбИРађРађРађРађРађРађ
РађРаИРБ┐РБДРађРБЙРАЪРАиРбцРБцРБцРа┤РБХРБХРБХРаХРБХРБХРБќРБ▓РањРбњРБњРБХРБХРБиРБЙРањРБХРб▓РањРа▓РБ┐РаЄРађРбИРађРаёРБЙРбИРађРађРађРађРађРађ
РађРађРа╣РБ┐РАёРбИРАЄРб╣РАёРађРб╣РађРбИРАђРбЙРађРбИРБ┐РБ┐РБ┐РБ┐РБ┐РБ┐РБ┐РБ┐РА┐Рб╗РађРБ┐РбИРАёРађРАъРађРађРбИРбђРАёРБ┐Рб║РАєРађРађРађРађРађ
РађРађРађРаЎРБ┐РађРАЄРаљРБЄРађРбИРађРаѕРБЄРбИРБЄРБгРБ┐РБ┐РБ╣РБ┐РБ┐РБ┐РБ┐РаЪРађРбИРБцРБЈРбИРаєРбаРАЄРађРађРбИРађРаЂРб»РбИРБиРађРађРађРађРађ
РађРађРађРађРБ┐РАєРБДРађРа╣РАёРаўРАЄРађРБ┐РБ┐РБ┐РБ┐РАЪРб┐РБ┐РБ┐Ра┐РаЪРаЂРађРађРаИРБ┐РБ┐РБ┐РБђРБЙРађРађРађРаИРађРађРаИРАќРб╣РАђРађРађРађРађ
РађРађРађРађРб╗РађРБ┐РађРађРбДРбаРБДРБ╝РБ┐РБ┐РБ┐РаЏРаЃРађРађРађРађРађРађРађРађРађРаЅРб╣РБ┐Рб╣РБ┐РБХРБХРБцРБђРАђРађРађРб╗РаўРАЄРађРађРађРађ
РађРађРађРађРбИРађРБ╣РБаРБ┤РБЙРБЙРБ┐РБ┐РБ┐РБ┐РБ┐РБЄРађРађРађРађРађРађРађРађРађРађРбаРБ┐РА┐РБИРБ┐РБ┐РБ┐РБ┐РБ┐РБ┐РБ┐РБХРБЙРБцРБиРађРађРађРађ
РађРађРБђРБаРаЙРБ┐РБ┐РБ┐РБ┐РБ┐РБ┐РБ┐РБ┐РБ┐РБ┐РБ┐РБ┐РБЄРађРбђРБђРБђРБцРБёРБђРађРађРБЙРБ┐РБиРБ┐РБ┐РБ┐РБ┐Ра┐Рб┐РБ┐РБ┐РБ┐РБ┐РБйРБ┐РБ┐РБиРБцРАђ
РБ┤РБ┐РБ┐РБ┐РБиРБХРБцРБцРБїРБЎРБЏРА╗Ра┐РБ┐РБ┐РБ┐РБ┐РБ┐РАЪРбЏРБІРБЅРБЂРБѕРАЏРА┐РБЙРБ┐РБ┐РБ┐РБ┐РаЪРБІРБ┤РБ┐РБ┐РБ┐РБ┐РБ┐РБ┐РБ┐РБ┐РБ┐РБ┐РБ┐РАЄ
```

сЂЊсѓїсЂ»сђЂжБЙсѓЅсЂфсЂёсѓбсЃЌсЃГсЃ╝сЃЂсЂеУфат«ЪсЂфУфъсѓітЈБсЂДсЂіт▒ісЂЉсЂЎсѓІсђЂсЂћсЂЈТЎ«жђџсЂ«сЃќсЃГсѓ░сЂДсЂЎсђѓ


## сЂфсЂюсЂЊсЂ«сЃќсЃГсѓ░сЂ»сЂЊсѓЊсЂфсЂФтцЅсЂфсѓЊсЂДсЂЎсЂІ№╝Ъ

сЂЊсЂ«сЃЄсѓХсѓцсЃ│сЂ»сЂѓсЂѕсЂджЂЋсЂёсѓњтЄ║сЂЌсЂдсЂёсЂЙсЂЎсђѓсЂесЂёсЂєсЂ«сѓѓсђЂУдќУдџуџёсЂфтђІТђДсЂ»т╝исЂЈт┐ЃсЂФТ«ІсѓІтЇ░У▒АсѓњућЪсѓђсЂеС┐АсЂўсЂдсЂёсѓІсЂІсѓЅсЂДсЂЎсђѓсЂесЂ»сЂёсЂѕсђЂтъІуа┤сѓісЂфУдІсЂЪуЏ«сЂ«УБЈтЂ┤сЂДсѓѓсђЂСИ╗ую╝сЂ»сЂѓсЂЈсЂЙсЂДСИГУ║ФРђЋРђЋсЂЎсЂфсѓЈсЂАсђЂсЂЙсЂБсЂЎсЂљсЂфТђЮсЂёсЂесђЂУфГУђЁсЂ«т┐ЃсЂФУДдсѓїсђЂсѓцсЃ│сѓ╣сЃћсЃгсЃ╝сѓисЃДсЃ│сѓњСИјсЂѕсѓЅсѓїсѓІС╝ЮсЂѕТќ╣сЂФсЂѓсѓісЂЙсЂЎ (╦Хрхћ рхЋ рхћ╦Х)


## сЂѓсЂфсЂЪсЂ«ТіЋуе┐сѓњсЂЎсЂ╣сЂдУдІсѓІсЂФсЂ»сЂЕсЂЊсѓњУдІсѓїсЂ░сЂёсЂёсЂДсЂЎсЂІ№╝Ъ

сЂЊсЂ«сЃќсЃГсѓ░сЂ«сЂЎсЂ╣сЂдсЂ«сѓ│сЃ│сЃєсЃ│сЃёсѓњТјбу┤бсЂЌсЂЪсЂёТќ╣сЂ«сЂЪсѓЂсЂФсђЂтљёсЃџсЃ╝сѓИсЂ«сЂЎсЂ╣сЂдсЂ«УеўС║ІсѓњсЂЙсЂесѓЂсЂЪсѓбсЃ╝сѓФсѓцсЃќсЃџсЃ╝сѓИсѓњућеТёЈсЂЌсЂЙсЂЌсЂЪсђѓТќ░сЂЌсЂёТіЋуе┐сЂ»тЁгжќІсЂЋсѓїсѓІсЂЪсЂ│сЂФУЄфтІЋуџёсЂФУАеуц║сЂЋсѓїсЂЙсЂЎсђѓсѓ╣сЃасЃ╝сѓ║сЂФТЊЇСйюсЂДсЂЇсѓІсѓѕсЂєсђЂтЈ│СИІсЂ«уЪбтЇ░сЃюсѓ┐сЃ│сѓњсЂћтѕЕућесЂЈсЂасЂЋсЂёсђѓуЏ┤ТёЪуџёсЂФсЃісЃЊсѓ▓сЃ╝сЃѕсЂДсЂЇсЂЙсЂЎсђѓ


## сЃќсЃГсѓ░сѓ│сЃ│сЃЮсЃ╝сЃЇсЃ│сЃѕсЂФсЂцсЂёсЂд

сЂЊсЂ«сЃќсЃГсѓ░сЂ»сђЂсЃЄсѓХсѓцсЃ│сЂеТЕЪУЃйсЂ«УдЂС╗ХсЂФтљѕсѓЈсЂЏсЂдсѓФсѓ╣сѓ┐сЃъсѓцсѓ║сЂЌсЂЪ Cayman Custom сЃєсЃ│сЃЌсЃгсЃ╝сЃѕсѓњСй┐ућесЂЌсЂдсЂёсЂЙсЂЎсђѓсЃЊсѓИсЃЦсѓбсЃФУдЂу┤асЂФсЂцсЂёсЂдсЂ»сђЂУЃїТЎ»ућ╗тЃЈсЂ«тцДтЇісѓњ Pixiv сЂІсѓЅСй┐ућесЂЌсЂдсЂёсЂЙсЂЎсЂїсђЂсЂ╗сЂІсЂ«тЄ║ТЅђсЂ«сѓѓсЂ«сѓѓСИђжЃетљФсЂЙсѓїсЂЙсЂЎсђѓсЂЎсЂ╣сЂдсЂ«сѓ»сЃгсѓИсЃЃсЃѕсЂетЄ║тЁИТЃЁта▒сЂ»сђЂсЃАсѓцсЃ│сЂ« GitHub сЃфсЃЮсѓИсЃѕсЃфсЂДт«їтЁесЂФтЁгжќІсЂЌсЂдсЂёсЂЙсЂЎсђѓ


## т░єТЮЦсЂФтљЉсЂЉсЂЪуЏ«ТеЎ

сЂєсЃ╝сѓЊРђдуЏ«уџёсЂФсЂцсЂёсЂдсЂ»сЂЙсЂаТђЮсЂёсЂцсЂёсЂдсЂёсЂЙсЂЏсѓЊсЂїсђЂС╗ісЂ«сЂесЂЊсѓЇсЂ»сѓхсѓцсЃѕсЂ«сЃЄсѓХсѓцсЃ│сѓњтцЅТЏ┤сЂЎсѓІсЂЊсЂесЂЌсЂІсЂДсЂЇсЂџсђЂсЂЊсЂАсѓЅсЂДУЄфтѕєсЂ«сЃфсЃЮсѓИсЃѕсЃфсѓѓтбЌсѓёсЂЌсЂдсЂёсЂЈсЂцсѓѓсѓісЂДсЂЎсђѓ

## сѓйсЃ╝сѓисЃБсЃФсЃАсЃЄсѓБсѓбсЂ«сЃфсЃ│сѓ»№╝џ

>_тљЏсЂ»сЂЇсЂБсЂесђЂС┐║сЂ«сЂЊсЂесѓњУф┐сЂ╣сЂЪсѓісЂЌсЂфсЂёсЂЈсѓЅсЂёсЂФсЂ»У│бсЂёсѓѕсЂГ Я┤дЯхЇЯ┤дЯ┤┐(╦х Рђб╠ђ р┤Ќ - ╦х ) РюД_
